PostgreSQL 18 is the selected database. Production migration files do not exist
yet; the API repository currently creates the report history table idempotently
when the Docker runtime starts or a report repository operation runs.

## `idea_reports`

| Column | Type | Notes |

| `research_requested` | `boolean` | true when the report used the search-and-organization path |

| `report` | `jsonb` | full `IdeaReportResponse` payload used by the detail page |
